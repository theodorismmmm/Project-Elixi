## 📖 About

**Project Elixi** is an open-source initiative to bring full **Fortnite** support to Linux. The goal is to provide a seamless, native experience for Linux gamers who want to enjoy Fortnite without relying on workarounds, Wine, or virtual machines.

---

## ✨ Planned Features

- 🐧 **Native Linux Support** — Run Fortnite directly on major Linux distributions
- ⚡ **High Performance** — Optimised for minimal overhead and maximum frame rates
- 🔧 **Easy Setup** — Simple installation process for beginners and power users alike
- 🛡️ **Anti-Cheat Compatibility** — Working towards full compatibility with Epic Games' anti-cheat systems
- 🖥️ **Wide Distro Support** — Targeting Ubuntu, Fedora, Arch Linux, and more
